## 2. Le Code Python (Laboratoire)

Ce script illustre comment charger, nettoyer, explorer et visualiser les tendances d‚Äôun march√©.

```python
import numpy as np

import matplotlib.pyplot as plt

# --- PHASE 1 : IMPORTATION DES DONN√âES ---

# --- PHASE 2 : NETTOYAGE ---
```
